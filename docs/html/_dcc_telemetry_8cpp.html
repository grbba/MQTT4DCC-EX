Detailed Description

Telemetry class allowing to report metrics to a specific MQTT topic for monitoring/debugging etc. Either, by adding a metric, this class when it's loop is getting called reoprts all the available metric or it can be called individually. Two specific measurements are provided concerning memory consumption and time measurements.
